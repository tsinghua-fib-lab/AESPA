# Data Directory Structure

## Required Data

Place your data in the following structure:

```
data/
├── lst/
│   ├── train/
│   │   ├── tract_id_1.npy  # LST values [day_temp, night_temp]
│   │   └── ...
│   ├── val/
│   └── test/
├── street/
│   ├── train/
│   │   ├── tract_id_1/
│   │   │   ├── img1.jpg
│   │   │   └── ...
│   │   └── ...
│   ├── val/
│   └── test/
├── satellite/
│   ├── train/
│   │   ├── tract_id_1.png
│   │   └── ...
│   ├── val/
│   └── test/
├── mobility/
│   ├── train/
│   │   ├── tract_id_1.npy  # Shape: [168, 3] (weekly mobility patterns)
│   │   └── ...
│   ├── val/
│   └── test/
└── concepts/  # Generated by preprocess_proxies.py
    ├── train/
    │   ├── tract_id_1_ndvi_proxy.npy
    │   ├── tract_id_1_tree_canopy_proxy.npy
    │   ├── tract_id_1_impervious_proxy.npy
    │   └── tract_id_1_albedo_proxy.npy
    ├── val/
    └── test/
```

## Data Format

- **LST**: NumPy array with shape `(2,)` containing `[day_temp, night_temp]` in Celsius
- **Street View**: Directory containing multiple RGB images (JPG/PNG)
- **Satellite**: Single RGB image (PNG) for each tract
- **Mobility**: NumPy array with shape `(168, 3)` representing weekly patterns (visit/outbound/stay)
- **Concepts**: NumPy scalar files for each proxy indicator

## Preprocessing

Run the proxy preprocessing script:

```bash
python data/preprocess_proxies.py --data-dir /path/to/data --split train
python data/preprocess_proxies.py --data-dir /path/to/data --split val
python data/preprocess_proxies.py --data-dir /path/to/data --split test
```

This will generate the `concepts/` directory with physical proxy indicators computed from street view images.
